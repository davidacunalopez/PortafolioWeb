Translucent glass container for bento tiles, project cards and feature panels. Lifts + glows on hover by default.

```jsx
<Card>
  <div style={{fontSize:'var(--fs-label)',color:'var(--accent)'}}>Backend</div>
  …
</Card>
<Card accent>Compromiso con cada proyecto</Card>
```

Props: `accent` (soft gradient wash), `hover` (default true), `padding`.
